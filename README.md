# Business Process Analysis - BPIC 2017 Event Log
This repository contains the complete analysis, process models, advanced analysis charts and the final report for the analysis of the BPIC 2017 Event Log. The project was conducted as a part of Business Process Prediction, Simulation and Optimization course at Technical University of Munich.

Repository Structure:
* **notebook.ipynb** : The main Jupyter Notebook containing the end-to-end data science analysis:
    * **Preporcessing**
    * **Basic Data Analysis**
    * **Process Discovery with Alpha, Heuristic and Inductive Miner**
    * **Quality Metric Analysis with Fitness, Precision, Generalization and Simplicity for the Event Log**
    * **Quality Metric Analysis with Fitness, Precision, Generalization and Simplicity for the Recreated BPMN**
    * **Advanced Analysis: Attribute-Driven Segmentation and Time-Series Analysis**
